先上快捷复制模板！（不包含新页面原有的title、date、tags）

Post Front-matter

  • 文章页markdown页眉配置参数样式一栏
优先顺序 配置 解释
01 title 【必需】页面标题
02 date 【必需】页面创建日期
03 tags 【必需】标籤
04 categories 【可选】分类
05 abbrlink 【可选】所在网页子名
06 description 【可选】页面描述
07 swiper_index 【可选】首页轮播设置（参数为正数，数值大者先展示）
08 cover 【可选】页面在主页显示的封面(内页无图)
09 comments 【可选】显示页面评论模块 (默认 true)
10 aside 【可选】显示侧边栏 (默认 true)
11 keywords 【可选】页面关键字
12 aplayer 【可选】在需要的页面加载 aplayer 的 js 和 css, 请参考文章下面的音乐配置
13 mathjax 【可选】显示 mathjax (当设置 mathjax 的 per_page: false 时，才需要配置，默认 false)
14 katex 【可选】显示 katex (当设置 katex 的 per_page: false 时，才需要配置，默认 false)
