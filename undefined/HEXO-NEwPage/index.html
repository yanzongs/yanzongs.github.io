书写流程

第一步 创建新篇

  1. 通过在博客根目录下使用Git Bash中的指令

    hexo new post "<新的名章名称>"创建新的文章

  2. 通过在博客根目录下使用Git Bash中的指令

    hexo new page "<新的名章名称>"创建新的主页

添加文章摘要

  • 在文章中使用 <!-- more -->，那么 <!-- more --> 出现之前的文字将会被视为摘要

  • 首页文章卡片中将只出现这部分文字，同时这部分文字也会出现在正文之中

如果在Font-matter中设置了description，那么在首页的文章轮播中将会显示description的内容而不是此摘要


相对路径引用

  1. 在[Blogroot]/source/_posts下会产生与文章同名的文件夹（如果没有就自己创建）

  2. 将本地文件放入同名文件夹中等待引用

  3. 使用指令{% [类型] [文件名] [其他参数] %}进行相对路径引用

参数 用法
类型 【必选】 pdf、word、img等等
文件名 【必选】 文件夹下文件名（记得加后缀）
其他参数 【可选】 如：width=<数字>、heigth=<数字>等
